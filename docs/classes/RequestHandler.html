request

  • request(method: requestMethod, route: string, requireAuthentication: boolean, content?: Object): Promise<[any, any]>
    • Defined in Util/RequestHandler.ts:51

    Makes desired HTTP request to Discord Rest API. Remember to handle errors by yourself, it will return array with output on first place (0) and potential error on second slot (1). Second slot will be null if request was successfull and there's no errors. First slot will be null if there was any error catched.

    example
    const [result, error] = await restHandler.request('GET', '/gateway/bot', true)

    Parameters

    • method: requestMethod
    • route: string
    • requireAuthentication: boolean
    • Optional content: Object
